# Tab Commands

* `:tabnew [file]` – Open a new tab (optionally with a file).
* `:tabedit [file]` (`:tabe`) – Same as `:tabnew [file]`.
* `:tabclose` (`:tabc`) – Close the current tab.
* `:tabonly` (`:tabo`) – Close all other tabs except the current one.
* `:tabnext` (`:tabn`) – Go to the next tab.
* `:tabprevious` (`:tabp`) – Go to the previous tab.
* `gt` – Normal mode shortcut: next tab.
* `gT` – Normal mode shortcut: previous tab.
* `#gt` – Jump to tab number `#`. Example: `2gt` goes to tab 2.
* `:tabfirst` – Go to the first tab.
* `:tablast` – Go to the last tab.
* `:tabmove [N]` – Move the current tab to position `N`.
* `:tabs` – List all tabs with buffer info.
